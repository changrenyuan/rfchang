# 传输线理论

传输线是射频电路的基础。理解传输线理论是成为一名合格射频工程师的第一步。

## 什么是传输线？

传输线是用于引导电磁波传播的导体系统。在低频电路中，我们通常忽略导线的电气特性，但在射频和微波频段，导线的长度变得与信号波长相当，此时必须考虑传输线效应。

常见的传输线类型包括：

- **同轴线**：由内导体、外导体和中间的介质层组成，广泛应用于射频系统
- **微带线**：PCB 上最常见的传输线，结构简单，易于集成
- **带状线**：微带线的变体，两个地平面之间夹着一个信号层
- **共面波导**：地平面和信号导体在同一平面，适合高频应用

## 特性阻抗

传输线的特性阻抗（Characteristic Impedance，记作 $Z_0$）是传输线最重要的参数之一。它定义为：

$$
Z_0 = \sqrt{\frac{R + j\omega L}{G + j\omega C}}
$$

其中：
- $R$：单位长度电阻
- $L$：单位长度电感
- $G$：单位长度电导
- $C$：单位长度电容

对于无损耗传输线（$R=0, G=0$）：

$$
Z_0 = \sqrt{\frac{L}{C}}
$$

### 常见的特性阻抗值

- **50Ω**：最常用的特性阻抗，广泛应用于射频和微波系统
- **75Ω**：主要用于有线电视和视频传输
- **100Ω**：差分信号常用

## 反射系数

当信号从特性阻抗为 $Z_0$ 的传输线传输到负载阻抗 $Z_L$ 时，会发生反射。反射系数 $\Gamma$ 定义为：

$$
\Gamma = \frac{Z_L - Z_0}{Z_L + Z_0}
$$

### 特殊情况

- **匹配负载**（$Z_L = Z_0$）：$\Gamma = 0$，无反射
- **开路**（$Z_L = \infty$）：$\Gamma = 1$，全反射，同相
- **短路**（$Z_L = 0$）：$\Gamma = -1$，全反射，反相

## 驻波比（VSWR）

电压驻波比（Voltage Standing Wave Ratio，VSWR）是衡量匹配程度的重要参数：

$$
\text{VSWR} = \frac{1 + |\Gamma|}{1 - |\Gamma|}
$$

### VSWR 的物理意义

- **VSWR = 1**：完美匹配，无反射
- **VSWR > 1**：存在反射，匹配不良
- **VSWR → ∞**：完全失配（开路或短路）

### 实际应用中的 VSWR 要求

- **一般通信系统**：VSWR < 1.5
- **高精度测量系统**：VSWR < 1.1
- **实验室测试**：VSWR < 1.2

## 传输线的等效电路

传输线可以用分布式参数模型来描述：

```
---L---C---L---C---L---C---
|     |   |     |   |     |
R     G   R     G   R     G
|     |   |     |   |     |
---L---C---L---C---L---C---
```

每个微小长度 $\Delta x$ 的传输线包含：
- 串联电阻 $R \cdot \Delta x$
- 串联电感 $L \cdot \Delta x$
- 并联电导 $G \cdot \Delta x$
- 并联电容 $C \cdot \Delta x$

## 实际工程考虑

### 1. 微带线设计

微带线的特性阻抗取决于：
- 线宽 $W$
- 基板厚度 $h$
- 介电常数 $\varepsilon_r$

可以使用在线计算工具或专业软件（如 ADS、HFSS）进行精确计算。

### 2. 传输线长度

当传输线长度超过 $\lambda/10$ 时，必须考虑传输线效应。在 2.4GHz 时，$\lambda/10 \approx 12.5$mm。

### 3. 阻抗匹配设计

实际电路中，负载阻抗往往不等于传输线特性阻抗。需要通过匹配网络将负载阻抗变换到 $Z_0$。

传输线理论是射频工程的基础。掌握以下要点：

1. ✅ 理解特性阻抗的定义和物理意义
2. ✅ 掌握反射系数和 VSWR 的计算方法
3. ✅ 了解不同类型传输线的应用场景
